Gauss Seidel

Metodo Numericos

1¿Que es y para que nos sirve?

El metodo de gauss seidel es un metodo muy usando para poder resolver ecuaciones de primer grado usando un grind base de 3x3, por lo que este metodo nos proporciona una forma bastante sencilla y rapida de poder resolver sistemas 3x3 de ecuaciones donde el uso de otros metodos (como sustitucion, igualacion, etc) nos podrian llevar mas tiempo o nos dan un resultado incorrecto o menos preciso. La cosa con el metodo Gauss Seidel, es que tiende a manejarse mediante iteraciones, es decir, el uso de una operacion multiples veces. ¿Que quiere decir esto? Bueno, el uso de gauss seidel hace que los resultados sean aproximaciones, no valores exactos, lo cual hace que esto sea la unica desventaja de usar este metodo contra otros similares para la resolucion de sistemas 3x3. La ventaja de este metodo es que la seguir la misma operacion para poder resolver estos sistemas, tiende a ser programable con mucha facilidad, por lo que el uso de sistemas para poder obtener el resultado usando este metodo es bastante eficaz. A continuacion, les mostramos este pequeño prototipo de sistema que, similar a otros sistemas, puede resolver ecuaciones 3x3 usando este metodo, y aunque podria no se el mejor, es bastante sencillo de usar.
